# Health & Wellness Site

A modern website for health and wellness tips, articles, and resources. Features:
- Responsive design
- Blog, events, and resource pages
- Image gallery
- Appointment booking

## Folder Structure
- `css/` - Stylesheets
- `img/` - Images
- `js/` - JavaScript files
- `pages/` - HTML pages

## Getting Started
Open `index.html` in your browser to view the site.
